У цьому zip архіві є сам файл вебсайту та 4 додаткові файли з навченими нейромережами та історією навчання. Також є два пайтон файли, які навчали ці моделі. 
Щоб программа працювала можно або скопіювати 4 додаткові файла до директорії з основним, або запустити обидва скріпта, щоб вони створили ці файли.
